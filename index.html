Assistente Virtual com Python

Nesse curso, vamos abordar desde a parte básica, como Instalação e Configuração do Python, até a instalação de bibliotecas externas em PLN (Processamento de Linguagem Natural), que vai nos ajudar a realizar a Sintetização de Voz em Python e o Reconhecimento de Voz para que possamos criar o nosso assistente virtual. Vamos trabalhar com as seguintes bibliotecas: Espeak, Pyttsx3 e GTTS (Google Text To Speech), para Sintetização de Voz e Speech Recognition para trabalhar com Reconhecimento de Voz. No projeto final do curso, vamos ser capazes de criar um assistente virtual do zero, onde integraremos a Sintetização de Voz e o Reconhecimento de Voz, de modo que vamos automatizar determinadas operações num computador.

Desenvolvimento Web com Flask

Nos últimos tempos a área de Desenvolvimento Web tem crescido bastante, devido à grande demanda de criação de aplicativos e sites. Nesse curso você vai aprender a desenvolver aplicações web utilizando uma das linguagens que mais cresce a cada ano, a linguagem Python. No curso apresentamos o framework Flask que nos permite criar aplicações web completas, além de permitir uma fácil integração com serviços cognitivos, como Processamento de Linguagem Natural e Visão Computacional.Nesse curso, vamos abordar desde a parte básica, como Instalação e Configuração do Python, até a instalação de bibliotecas externas, como o Flask que a utilizaremos ao decorrer do curso.

Desenvolvendo Data Apps com Streamlit

O Streamlit é uma biblioteca Python de código aberto que facilita a criação e o compartilhamento de aplicativos da Web personalizados e bonitos para aprendizado de máquina e ciência de dados que podem ser usados ​​para compartilhar resultados de análise, criar experiências interativas complexas e ilustrar novos modelos de aprendizado de máquina. Em apenas alguns minutos, você pode criar e implantar aplicativos de dados poderosos. Nesse curso quero apresentar essa incrível ferramenta a você, que gosta da área de Data Science e que deseja evoluir em sua carreira, aprendendo a utilizar aplicações web para servir de interface com o modelo preditivo que foi criado. A você que gosta da área de Desenvolvimento Web, quero apresentar a você, uma ferramenta que vai te ajudar na criação de Data Apps, com a possibilidade de criar aplicações analíticas focadas em Machine Learning.
