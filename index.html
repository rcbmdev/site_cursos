LangChain - Desenvolva Agentes e Apps com LLM

Este curso oferece uma imersão prática no uso de IA, abordando desde modelos de linguagem da OpenAI, como GPT-4, até ferramentas avançadas como DALL·E e Whisper. O curso ensina a criar chatbots personalizados, integrar sistemas com function calling e explorar imagens e áudio. A segunda parte foca no LangChain, facilitando a integração de LLMs em fluxos complexos, otimização de prompts e análise de saídas. Também cobre o uso de memória, recuperação de informações (RAG) e estratégias de depuração. Por fim, são abordadas práticas avançadas para construir agentes autônomos, otimizar fluxos e gerenciar recursos, com estudos de caso e aplicações reais.

Técnicas de Invasão com Kali Linux

O trabalho de um Pentest consiste inicialmente na coleta de informações. Nesse curso, dividiremos essa etapa em duas: Informações de Negócio e Informações de Infraestrutura. Abordaremos cada uma em uma seção diferente. Em informações de negócio, aprenderemos sobre Google Hacking, Wayback Machine, TheHarvester, Metagoofil, etc. Já em informações de infraestrutura, aprenderemos sobre Whois, Host, Shodan, Recon-Ng, Whatweb, etc. Ao final destas seções, você terá adquirido muito conhecimento e poderá aplicar na prática para coletar informações do seu sistema alvo.

Desenvolvendo Dashboards no Python com Streamlit e Dash

Desenvolvimento Web com Django

Neste curso, você aprenderá a desenvolver aplicações web poderosas e dinâmicas utilizando o Django, um dos frameworks mais populares para Python. Com suporte dedicado e aulas atualizadas, você sairá preparado para criar aplicações web modernas e escaláveis, pronto para destacar-se no mercado de desenvolvimento web.
